## 制作の指針

- 背景は透過PNGを推奨
- 小さく表示しても読めるシルエットにする
- 深いネイビー、ゴールド、エメラルド、白い光を基調にする
- 子どもっぽくなりすぎない、大人向けRPGの質感にする

このフェーズではアプリ内へのバッジ差し替えはまだ行いません。
