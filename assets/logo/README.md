# Growth Quest — Logo Marks

このフォルダには、画面内に直接表示するロゴ画像を配置します。
PWA アイコン（`assets/icons/`）とは別物です。

## 配置するファイル

```text
logo-mark-64.png                64×64    ヘッダー左側のロゴマーク（テキスト無し）
growth-quest-logo-horizontal.png       横長 ローンチ画面や About 用
```

## 制作の指針

- **ヘッダーマーク（logo-mark-64.png）**:
  - 透過背景
  - シアン `#06b6d4` ベース、ホワイトハイライト
  - ⚔ の意匠を 18〜24px 表示でも読める強さで
  - 文字は含めない（隣にテキストロゴを並べる前提）

- **横長ロゴ（growth-quest-logo-horizontal.png）**:
  - 透過背景
  - 「⚔ Growth Quest」のテキスト＋マーク
  - シアン→紫→レッドのグラデーション（既存ローンチ画面と合わせる）
  - ローンチ画面・設定の「アプリについて」で使用予定

`index.html` のヘッダーとローンチ画面は `logo-mark-64.png` を参照する設定になっています。
画像が未配置の場合は、画面崩れを避けるため画像だけ非表示になります。
