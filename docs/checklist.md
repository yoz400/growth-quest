# Growth Quest 動作確認リスト

このリストは、Growth Quest を直したあとに「大きく壊れていないか」を確認するためのものです。

## 使い方

変更後に `index.html` をブラウザで開き、上から順番に確認します。

```text
コードを変更する
  ↓
ブラウザで開く
  ↓
このリストで確認する
  ↓
問題があれば直す
```

## 基本表示

- [ ] `Growth Quest` の画面が開く
- [ ] 背景や文字色が崩れていない
- [ ] ヘッダーのボタンが表示されている
- [ ] アバター画像が表示されている
- [ ] 装備画像が読み込まれている
- [ ] スマホ幅でも画面が横にはみ出さない

## タイマー

- [ ] 開始ボタンでタイマーが動く
- [ ] 一時停止または停止ができる
- [ ] 学習完了後にXPが増える
- [ ] 休憩表示が出る
- [ ] 通知や音の設定を変えても画面が壊れない

## 学習データ

- [ ] XPとレベルが表示される
- [ ] 累計学習時間が表示される
- [ ] 連続記録が表示される
- [ ] ブラウザを再読み込みしてもデータが残る

## すごろく

- [ ] すごろく画面を開ける
- [ ] 学習後にマスが進む
- [ ] イベントマスの結果が表示される
- [ ] アイテムや装備の獲得表示が出る

## 装備とアバター

- [ ] 装備画面を開ける
- [ ] 所持装備が表示される
- [ ] 装備する、外す操作ができる
- [ ] アバター画面に装備状態が反映される

## 週間レビュー・バッジ・スキル

- [ ] 週間レビュー画面を開ける
- [ ] バッジ画面を開ける
- [ ] スキルツリー画面を開ける
- [ ] 各画面を閉じられる

## 保存の確認

- [ ] 設定変更後、再読み込みしても設定が残る
- [ ] ウィジェットの並び替え後、再読み込みしても順番が残る

## 重要な用語

- **再読み込み**: ブラウザで同じページをもう一度読み直すことです。
- **表示崩れ**: 文字やボタンが重なったり、画面外にはみ出したりすることです。
- **動作確認**: 変更後に、機能が壊れていないか実際に試すことです。
