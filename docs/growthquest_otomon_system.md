## 目的

Growth Questに **「オトモン」** という相棒モンスターシステムを追加する。
オトモンは、単なるペットやコレクション要素ではなく、**学習・習慣化・休憩・睡眠・自己管理を楽しく続けるためのナッジ役**として設計する。

> **ナッジ（nudge）とは:** 「そっと後押しする」という意味。命令や強制ではなく、自然に良い行動を選びたくなるやさしいきっかけを与える仕組み。

Growth Questのコア思想は **「楽しくをモットーに自信を成長させる」**。
そのためオトモンシステムも、プレイヤーを追い詰めるものではなく、現実の小さな行動を楽しく発生させ、自信につなげる仕組みにする。

| 用語 | 意味 |

| 目覚めアイテム | 卵に使うことでクエストを発生させるアイテム |

### 命名規約（コード上のID）

> **コード上の識別子（ID）は `otomon` を使う。** 既存コードでは `pet` は「装備カテゴリ（見た目）」、`companion` は「装備の絆時間」として既に使用中のため、衝突を避けて新しい名前空間 `otomon` を割り当てる。

**原則：データ上のIDは英語 snake_case、ユーザーに見せる名前は日本語。**

| 区分 | 例 |
|---|---|
| 個体ID / 表示名 | `echo_slime` / こだまスライム |
| 卵ID / 表示名 | `echo_egg` / こだまの卵 |
| 目覚めアイテムID / 表示名 | `echo_flute` / こだまの笛 |
| 旅先ID / 表示名 | `cave` / 洞窟 |
| 属性ID / 表示名 | `study` / 学習 |

## 基本ループ

```text
 ① 学習・ポモドーロ・デイリークエスト達成
        ▼
 ② 双六を回して旅をする
        ▼
 ③ 旅先で「オトモンの卵」を見つける
        ▼
 ④ 卵に「目覚めアイテム」を使う
        ▼
 ⑤ アイテムに応じて「オトモンクエスト」が発生
        ▼
 ⑥ 現実行動を達成（腕立て5回 / 笑顔で挨拶 / 5分学習 など）
        ▼
 ⑦ 達成 → 孵化ゲージが進む
        ▼
 ⑧ ゲージ満タン → オトモン誕生！
        ▼
 ⑨ 以後、その行動テーマを支援する相棒になる ──▶（次の周回へ）
```

## システムの重要思想

- アイテムは単なる報酬ではなく、**現実行動を発生させるトリガー**として扱う。
- 卵は**未来報酬・期待感**を生む要素。
- オトモンクエストは、**ゲーム内報酬と現実行動をつなぐ仕組み**。
- オトモンは孵化後、プレイヤーの**習慣化・学習・休憩・睡眠・自己管理を支えるナッジ役**になる。

## 目覚めアイテム

目覚めアイテムは、オトモンの卵に使用することでオトモンクエストを発生させる**消費アイテム**。
アイテムには属性があり、属性によって発生するクエストや生まれやすいオトモンが変わる。

### 基本の目覚めアイテム

| 属性 | 目覚めアイテム | 発生しやすいクエスト | 相性の良いオトモン |
|---|---|---|---|
| 学習 | こだまの笛、📑よみかけのしおり | 5分学習、音読、復習、1ページ読む | こだまスライム、ひびきコウモリ、ルーンとかげ、クリスタルラビット |
| 集中 | 集中のロウソク | 通知OFF、5分集中、1ポモドーロ | ランタンバット、まどろみクラゲ、ばちばちスパーク |
| 運動 | ちからの木の実 | 腕立て5回、スクワット5回、散歩 | まめドラコ、こぐまナイト、ちびタイタン、もぐらランサー |
| 回復 | ひだまりブランケット | 深呼吸、水分補給、休憩 | ひだまりモコ、ふわ雲ラム、あめだまスライム、きのこリス |
| 睡眠 | まどろみの羽 | 寝る準備、スマホを遠ざける | ねむりヒツジ、ねむけバク、すやすやノーム、うたたねドラゴン |
| 交流 | えがおの鈴 | 笑顔で挨拶、感謝を伝える | こんぺいとうフェアリー、しろたまユニコ、こはるアルパカ |
| 整理 | すっきり小ぼうき、🔧しゅうりキット | 机を1分片付ける、不要物を捨てる、散らかりを直す | からくりネズミ、ぷちミミック、ねじまきゴーレム、さびネジインプ |
| 発想 | ひらめきの貝殻 | アイデアを1つメモ、振り返り | ひらめきイルカ、そよ風フェレット、まよい蝶、くものこスフィンクス |
| 自制 | まよけのお守り | やらないことリスト確認、危険行動回避 | まよけフクロウ、くろねこシェイド、こまいぬベビー、まじないカラス |
| 冒険 🆕 | 🗺ぼうけんの地図、🧭たびのコンパス | 外に出て歩く、次の一歩を決める、新しいことに触れる | ほたるウルフ、すなネコマタ、すずめ天狗、ゆらめきウィスプ、かぜきりツバメ竜 |
| 生活回復 | しずくの小びん | 水を飲む、顔を洗う、保湿する | しずくセイレーン、あわあわタツノコ、かいがらマーメイド |

> **実装メモ（整合性）:** オトモンは上の **10の行動属性**（学習・集中・運動・回復・睡眠・整理・交流・発想・自制・**冒険**）のいずれかに分類し、その属性の目覚めアイテムで孵る。当初「冒険」属性にアイテムが無く冒険系オトモンが孵せなかったため、🗺ぼうけんの地図・🧭たびのコンパスを新設して全10属性をカバー。あわせて 📑よみかけのしおり（学習）・🔧しゅうりキット（整理）も追加。現在 **基本14 ＋ 特別5 ＝ 19アイテム**（`scripts/otomon.js` の `WAKE_ITEM_MASTER`）。

### 特別な目覚めアイテム

| アイテム | 効果 |
|---|---|
| はじまりの種 | どの卵にも使える。着手系クエストを発生させる |
| 自信のしおり | 自己肯定・振り返り系オトモンに有効 |
| にじいろの雫 | 進化分岐・レア個体に影響する |
| きずなのリボン | すでに仲間にしたオトモンとの親密度を上げる |
| 再挑戦の羽ペン | 未達成のオトモンクエストに再挑戦できる |

> **ロスターとの名寄せ例:** 🪶再挑戦の羽ペン ↔ みならいフェニックス（何度も復活）／ 🌈にじいろの雫 ↔ にじいろスライム（進化）／ 🌱はじまりの種 ↔ つゆくさピクシー（朝のやる気＝着手）。

## オトモンクエスト

オトモンの卵に目覚めアイテムを使用することで発生する特別なクエスト。
現実世界の小さな行動を達成すると孵化ゲージが進み、満タンで新しいオトモンが誕生する。

### 設計方針

- プレイヤーに過度な負担をかけない。基本は**30秒〜5分**で達成できる小さな行動。
- 高レアでも単純に負荷を重くしない。高レアは量ではなく、以下で難度を上げる：
  - 継続性
  - 意味の深さ
  - 自分との約束を守ること
  - 少し勇気がいる行動
  - 苦手タスクへの着手

### レア度ごとのクエスト難度

| レア度 | 難度 | クエスト例 |
|---|---|---|
| N | 30秒〜1分 | 水を飲む、深呼吸3回 |
| R | 1〜5分 | 腕立て5回、机を1分片付ける |
| SR | 5〜15分 | 5分学習、散歩5分、復習 |
| SSR | 1日単位 | 1セッション完了、夜に記録 |
| UR | 連続・特別 | 3日連続達成、苦手タスクに着手 |

> 注意：URでも「100回腕立て」のような根性ゲーにはしない。

### 失敗時の扱い

オトモンクエストに失敗しても**卵は割れない**。未達成の場合は **「卵が少し眠る」** 程度の演出にし、翌日以降に再挑戦できるようにする。プレイヤーの自信を折るペナルティは避ける。

## 導きの妖精について

Growth Questのナビゲーター的存在。

**役割:** 今日のおすすめクエストを1つ提案／迷ったときに最小行動を示す／ギルドやオトモンクエストを案内／失敗時に責めずに再開ルートを提示。

**性格:** 優しい／責めない／迷った時だけそっと出てくる／「次の一歩」を照らす。

**ビジュアル:** 金髪の小さな妖精／青い羽／光輪／ランタン／太めの濃い輪郭／高品質なピクセルアート風／表情差分（通常・喜び・驚き・悲しみ・ウインク・照れ）。

## こだまスライムについて

学習音に反応して跳ねるオトモン。

**役割:** 学習開始のナッジ／開始ボタンを押した瞬間に反応／学習を始めたこと自体を肯定。

**ビジュアル:** ミントグリーンの半透明スライム／音波アンテナ／体内にこだま核／音符や波紋のエフェクト（ただし64px実装時は情報量を抑える）。

```text
旅先で「こだまの卵」を見つけた！
```

```text
オトモンクエスト発生！

クエスト：5分だけ学習を開始しよう
```

```text
クエスト達成！
卵の中から小さな音が響いた。
こだまの卵が光りはじめた！

孵化ゲージ：1 / 3
```

```text
卵がぽよんと跳ねた！
こだまスライムが生まれた！
学習音に反応して跳ねる相棒だ。
```

### 未達成時

```text
こだまの卵は、少し眠ってしまった……
大丈夫。
また明日、もう一度起こしてあげよう。
```

## 実装時の注意

1. オトモンシステムはコレクションだけで終わらせない。
2. オトモンは習慣化・学習・休憩・睡眠・自己管理のナッジに紐づける。
3. アイテムは消費型の行動トリガーとして扱う。
4. 卵とオトモンクエストにより、双六と現実行動を接続する。
5. 失敗時にペナルティを強くしすぎない。
6. 「自信を育てる」方向に演出する。
7. 高レアほど負荷を増やすのではなく、意味のある行動・継続性・自己選択性を増やす。
8. 最初から100体を実装しようとせず、第1弾10体から始める。
9. まずはデータ構造とUI導線を整え、後からオトモン数を増やせる設計にする。
