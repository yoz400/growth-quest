## 1. 背景

召喚オンボーディングの「冒険者を選ぶ」画面（2026-07-03 実機評価で確認）:

1. **名前が無機質**:「冒険者A/B/C」。妖精・召喚の儀という世界観と不釣り合い
2. **選択状態が曖昧**: AとBの両方に枠が付いて見え、どれが選ばれているか分からない
3. アバター画面の切替ボタン（app.js 8681付近 `av-type-btn`）も「冒険者A 男性風・短髪」表記

## 2. 設計（2026-07-04 ヨージ決定を反映）

**基本コンセプト: 冒険者＝プレイヤー自身の分身。**
オンボーディングで入力する「冒険者名」（`playerName`, app.js 10273付近）を
アバターの名前としてそのまま使う。**未入力のときだけ**デフォルト名を出す。

### 2.1 名前の決定ロジック（データは互換維持）

保存値は今まで通り `avatarType: 'A' | 'B' | 'C'` ＋ `playerName`（既存のまま）。

```javascript
const ADVENTURERS = {
  A: { fallback: 'レン', title: '凜々しい剣士',   desc: 'まっすぐ突き進む努力家' },
  B: { fallback: 'ミア', title: '聡明な魔法使い', desc: 'コツコツ積み上げる知性派' },
  C: { fallback: 'ソラ', title: '旅する吟遊詩人', desc: '自由に楽しく続ける自由人' },
};

// 表示名はこの1関数に集約する（直書き禁止）
function adventurerName(type = avatarType) {
  const nm = (playerName || '').trim();
  return nm || ADVENTURERS[type]?.fallback || '冒険者';
}
```

- 適用箇所: 「冒険者A/B/C」を grep して全置換。表示は必ず `adventurerName()` を通す
- 召喚の選択カード（app.js 10380付近）は**名前入力より前**に出るので、
  カードには肩書き＋desc を表示（「凜々しい剣士」等。A/B/C表記は消す）
- 設定で冒険者名を変えたら表示名も追従する（関数集約により自動で満たされる）

### 2.2 選択状態の明確化（召喚カード）

```text
未選択:  枠なし・彩度そのまま・少し暗く(opacity .75)
選択中:  シアンの枠＋外側グロー＋右上に ✓ バッジ＋名前がシアンに
```

- 選択は常に**ちょうど1人**。初期状態から A を選択済みにする（現状踏襲）
- カードに `aria-pressed="true/false"` を付与（選択UIの読み上げ対応）
- CSS例: `.summon-char.selected { border:2px solid var(--cyan); box-shadow:0 0 18px rgba(103,232,249,.35); }`
  `.summon-char:not(.selected) { opacity:.75; }` ＋ `.summon-char-check`（✓バッジ）

### 2.3 選んだキャラ以外は非表示（ヨージ決定）

- **アバター画面の「アバタータイプ」3ボタン切替（app.js 8678付近 `av-type-btn`）を廃止**し、
  選択中の冒険者だけを表示する（名前＝`adventurerName()`＋肩書き）
- 逃げ道として**設定に「冒険者を変更」を1行だけ追加**（select で 剣士/魔法使い/吟遊詩人）。
  変更したら即アバターに反映
- 召喚画面の案内文（app.js 10328付近）
  「（あとから「アバター」画面でいつでも変えられます）」→
  「（あとから設定で変えられます）」に修正（嘘にならないように）

### 2.4 名前を体験に反映（小さく世界観を強める）

- 召喚完了画面のセリフ:「ようこそ、剣士ヨージ。」
  （肩書き＋`adventurerName()`。名前未入力なら「ようこそ、剣士レン。」）

## 4. テスト手順

1. localStorage クリア → 召喚でカードを順にタップ → 常に1枚だけ選択表示
2. 名前「テスト」を入力して完了 → アバター画面に「テスト」＋肩書きが表示される
3. localStorage クリア → 名前を**空のまま**完了 → デフォルト名（選んだキャラの
   レン/ミア/ソラ）が表示される
4. アバター画面に他のキャラが表示されていない
5. 設定 →「冒険者を変更」で別キャラに → アバターが変わり、名前は維持される
6. 既存データ（avatarType:'B'＋playerName:'ヨージ'）で開く → 「ヨージ」表示
