# AGENTS.md

このファイルは、Codex がこのプロジェクトで作業するときの案内です。

## プロジェクトの現状

このプロジェクトは **Growth Quest** という、学習をゲームのように続けるためのブラウザアプリです。

現在の中心ファイルは次の通りです。

```text
claude-practice/
├─ index.html              アプリの入口
├─ styles/app.css          見た目を決めるCSS
├─ scripts/app.js          動きを決めるJavaScript
├─ assets/equipment/       装備アイテム画像
├─ adventurer-*.png        アバター画像
├─ README.md               プロジェクト説明
└─ docs/checklist.md       動作確認リスト
```

## 初心者向けの用語

- **HTML**: 画面の骨組みを作るファイルです。
- **CSS**: 色、余白、配置など、見た目を決めるファイルです。
- **JavaScript**: ボタンを押したときの動きや、データ保存などを担当するファイルです。
- **localStorage**: ブラウザの中にデータを保存する仕組みです。サーバーなしで使えます。
- **Git**: 変更履歴を保存する仕組みです。失敗した変更を戻しやすくします。

```text
index.html
   ↓ 読み込む
styles/app.css  +  scripts/app.js  +  画像素材
   ↓
Growth Quest の画面が動く
```

## 開き方

`index.html` をブラウザで開くと動きます。

開発用サーバーを使う場合は、プロジェクト直下で次のように起動できます。

```sh
python3 -m http.server 8000
```

そのあと、ブラウザで `http://localhost:8000` を開きます。

## コラボレーションのルール

- ユーザーの名前はヨージです。Codex のことは「クロ」と呼んでいます。
- 説明はすべて **初心者向け** にわかりやすく書いてください。
- 専門用語が出てきたら、**必ずその意味を説明** してください。
- 重要な概念や仕組みを説明するときは、**図解（テキスト図・ASCII図）を使って視覚的に伝えてください**。

## 作業方針

- 変更前に、対象ファイルの役割を確認します。
- 既存の見た目や動きをなるべく保ちます。
- 大きな変更をしたら、`docs/checklist.md` を使って動作確認します。
- `index.html`、`styles/app.css`、`scripts/app.js` の関係を崩さないようにします。
